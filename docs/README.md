# IPTV Pro Android

> 专为小米电视优化的极致轻量 IPTV 播放器

## 快速开始

### 系统要求

| 项目 | 要求 |
|------|------|
| **Android版本** | 4.4+ (API 19+) |
| **RAM** | 最低 1GB（运行占用 40-70MB） |
| **存储** | 最低 10MB 可用空间 |
| **网络** | 局域网访问后端服务 |

### 安装包信息

- **APK大小**: 3-5MB
- **安装后占用**: < 10MB
- **运行时内存**: 40-70MB

### 后端依赖

本应用需要连接 `iptv-scanner-pro` 后端服务：

```
默认服务器地址: http://192.168.9.158:5950
```

后端部署：
```bash
cd /home/reyan/Projects/iptv-scanner-pro
docker-compose up -d
```

### 首次使用

1. 安装 APK 到小米电视
2. 确保电视与后端服务在同一局域网
3. 打开应用，自动加载上次播放列表
4. 若无播放列表，按【菜单】键进入扫描配置

---

## 核心功能

### 1. 全屏播放
- 启动即全屏播放上次频道
- 左右键切换频道
- 上下键调节音量

### 2. 频道列表
- 按【确定】键显示频道列表
- 网格布局，遥控器导航
- 显示频道名称和分辨率

### 3. IPTV源扫描
- 按【菜单】键进入扫描配置
- 支持多线程扫描（推荐50线程）
- SSE实时推送扫描进度

### 4. 播放列表管理
- 自动保存播放列表到本地
- 启动自动加载上次列表
- 支持清空/重新扫描

---

## 遥控器操作

| 按键 | 全屏播放 | 频道列表 | 扫描配置 |
|------|---------|---------|---------|
| **上/下** | 音量 +/- | 上下移动 | 选项切换 |
| **左/右** | 上个/下个频道 | 分类切换 | 数值调整 |
| **确定(OK)** | 显示列表 | 播放选中 | 确认/扫描 |
| **返回** | - | 返回播放 | 返回主界面 |
| **菜单** | 进入扫描 | 进入扫描 | - |
| **主页** | 退出应用 | 退出应用 | 退出应用 |

## 安全声明

⚠️ **本应用针对低配置设备做了极致优化**：

1. **APK体积仅3-5MB** - 不占用电视存储
2. **运行时内存40-70MB** - 1GB电视可流畅运行
3. **零第三方依赖** - 纯Android原生API
4. **多重安全防护**:
   - 启动时内存检测
   - 运行时内存监控
   - 全局异常捕获
   - 崩溃自动恢复（安全模式）
