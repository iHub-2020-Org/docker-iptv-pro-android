# IPTV Pro Android - QA 检查清单

## 安装前检查

- [ ] APK文件存在且未损坏
- [ ] APK大小 < 5MB
- [ ] 目标设备Android版本 >= 4.4 (API 19)
- [ ] 目标设备可用内存 > 1GB
- [ ] 后端服务 IPTV-Scanner-Pro 运行中 (192.168.9.158:5950)
- [ ] 电视与后端在同一网络

## 安装验证

- [ ] ADB连接正常
- [ ] 安装命令成功执行

- [ ] 应用能正常启动
